# Table book — итоговое обновление

Сборка объединяет полный проект сайта с последним hotfix навигации и синхронизации.

## Что включено

- блок «Книга рецептов» поднят почти к верхнему бару с небольшим адаптивным отступом;
- «Мои рецепты» и «Меню недели» расположены выше карусели;
- страны показаны в круговой карусели со стрелками, автопрокруткой и свайпом;
- сохранены старые изображения-заглушки из `assets/countries`;
- информация о стране находится в отдельном прямоугольном нижнем боксе;
- карточки стран, типов блюд и рецептов приведены к стилю карточек «Мои рецепты» и «Меню недели»;
- на ПК и мобильных устройствах работает цепочка «страна → тип блюда → рецепты»;
- кнопка возврата на мобильном экране рецептов возвращает к типам блюд;
- добавлены «Мне нравится», сердечки на карточках и в открытом рецепте;
- избранное синхронизируется с существующей таблицей Supabase `public.user_app_state` в `app_state.favorites`;
- локальное хранилище используется как персональный офлайн-кэш, разделённый по `user_id`;
- сохранены Realtime-обновления и отправка офлайн-изменений после восстановления сети;
- кнопка «Оборудование» открывает энциклопедию;
- статистический блок с количеством кухонь и рецептов скрыт.

## Установка

1. Сделайте резервную копию опубликованного проекта.
2. Скопируйте всё содержимое папки `Table_book` поверх текущей версии сайта.
3. Не меняйте Supabase URL и публичный ключ: сборка использует подключение из существующего `js/app.js`.
4. Убедитесь, что таблица `public.user_app_state` добавлена в публикацию `supabase_realtime`, если нужна синхронизация между открытыми устройствами без обновления страницы.
5. После публикации выполните жёсткое обновление страницы (`Ctrl+F5`).

Новая SQL-миграция не требуется: `favorites` сохраняется внутри существующего JSONB-поля `app_state`.

## Быстрая проверка

- ПК: видны три карточки карусели и стрелки; страна открывает типы блюд.
- Мобильный: карусель листается свайпом; после типа отображаются только его рецепты; «← Типы блюд» возвращает к категориям.
- Авторизованный пользователь: сердечко сохраняется после перезагрузки и появляется на другом устройстве.
- «Оборудование»: открывается экран энциклопедии.
