# Домашняя книга рецептов

Статическое веб-приложение с 71 рецептами. Откройте `index.html` в браузере.

Функции:
- поиск по названию, ингредиентам и шагам;
- фильтры по стране, типу блюда и сложности;
- карточки рецептов и модальные окна с подробными инструкциями;
- чекбоксы для шагов приготовления;
- список покупок в localStorage;
- печать / сохранение в PDF через браузер.


Обновление: каталог расширен дополнительными рецептами для категорий «Завтраки», «Закуски», «Гарниры» и «Горячие блюда». Для карточек рецептов добавлены отдельные иконки-заглушки по типу блюда.


Обновление: добавлены тематические обложки кухонь, режим «Избранное», тёмная тема, более иллюстративные карточки и подборка здоровых рецептов с пометкой 🌿.
